Intro

In this lab, we'll analyze some longer simulations. Let's practice what we've learned in Lab 1 and figure out a little more along the way. The Lab assignment is linked here.

One thing about the Terminal: The commands that we run in the terminal will only work if we type them at the "prompt." The prompt is the line before the blinking cursor that says "YourID@computer:present/working/directory$". If you are currently running a program in the Terminal, there won't be a prompt. If you don't want to wait, you can open another Terminal and type in another command there. Also, if you open the gedit program in the Terminal, you cannot use another command in the Terminal until you close gedit or use this trick: in the terminal, press Ctrl+Z, then type bg, and press enter. Now gedit is running in the background, and you can type in another command. (This works for other things besides gedit, too.)
